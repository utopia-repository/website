404 - File Not Found

If you're getting this error from a package download (.deb, .dsc), it may be because the relevant package has been removed.

If you're getting this error from a changelog link, it may be that the changelogs haven't been automatically generated yet (these get cleared on package updates due to https://github.com/smira/aptly/issues/572).
